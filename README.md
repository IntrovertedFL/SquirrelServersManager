**Edit the .env file before anything**

## Production
```console
docker compose up
```

## Developement
```console
docker compose up -f docker-compose.dev.yml up 
```

## Main features:
- Statistics & metrics of your servers
- Ansible playbooks management and execution
- Containers management (actions, updates)
- Automations
